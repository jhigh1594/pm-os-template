# PM Communication Writer

Use the **elite-copywriter** skill for professional copywriting and strategic communications.

**Invoke**: `@~/.claude/skills/elite-copywriter/`

## When to Use This Command

Use `/write` for any PM communication work:
- Executive updates and stakeholder alignment
- Feature announcements and product communications
- Strategic recommendations and decision memos
- Customer messaging and value propositions
- Sales enablement and competitive positioning

## Quick Reference for PM Context

The elite-copywriter skill will gather audience and domain context. For Planview/AgilePlace work, it can leverage:

**From Project Context:**
- ICP: Enterprise (500-10,000+ employees), Financial Services, Insurance, Logistics, Manufacturing, Technology
- Products: AgilePlace, OKRs, Roadmaps, DPD
- Key personas: Director of IT, VP of Engineering, CTO, Agile Program Managers, RTEs, Product Owners
- Existing tools: Jira (75%), Azure DevOps (60%)

**Available Frameworks** (from elite-copywriter):
- **Executives**: BLUF, Minto Pyramid (SCQA), MECE
- **Engineers/ICs**: Problem → Solution → Outcome, Jobs to Be Done
- **Customers**: Before/After/Bridge, PAS, StoryBrand
- **Stakeholders**: Narrative structure, Data + story
