# Weekly Competitive Brief

Generate a comprehensive weekly competitive intelligence brief for tracked competitors.

## Usage

```
/weekly-comp [--week YYYY-WXX] [--competitors atlassian,linear,aha,monday,jira-jpd]
```

## What It Does

1. **Collects** data from all competitor sources (changelogs, pricing, blogs)
2. **Analyzes** changes for strategic significance
3. **Synthesizes** into actionable intelligence brief
4. **Stores** brief in Knowledge base for historical tracking
